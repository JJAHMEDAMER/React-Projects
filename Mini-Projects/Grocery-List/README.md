# Grocery List

### React App

An app that helps you remember the items to shop for, even if you've left or lost your list on the way to the store.

## App Feature

1. Use React 18
2. Pure Css
3. Add items
4. delete Items
5. Edit Item
6. Read and write to local storage

### App Components

1. Generative Card
2. React Use State Hook
3. Input Bar
4. Edit Bar
5. Local Storage Api
